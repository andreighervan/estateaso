3. PSD Structure

Photoshop files are located in main_files/psd folder. All files are layered with clean structure.

Main groups are labelled with colours. Here is typical structure:

  • Header - contains logo, menu and address bar
  • Slider - for pages with slideshow, Map - for listings or Page Top Image - for other pages
  • Main content - one or more sections, depending of page
  • Sidebar - contains blog categories, vertical search, latest post and featured offers. Not on all pages
  • Footer - contains contact info, quick links, latest posts, newsletter and copyright info.
